AutoPM

추진계획서 주제만 넣으면 Rule-based 인터뷰 후 Multi-Agent(CrewAI)가 Markdown·PPT(python-pptx)까지 만드는 해커톤 MVP입니다.

참고: 화면(UI)은 Streamlit 기반이라 장시간 웹소켓이 필요합니다. Vercel 서버리스에는 Streamlit 전체를 그대로 올리기 어렵기 때문에, 이 페이지는 안내·랜딩용입니다. 실행은 아래 중 하나를 사용하세요.

저장소

GitHub

로컬 실행

  • pip install -r requirements.txt
  • cp .env.example .env 후 OPENAI_API_KEY 선택 입력
  • streamlit run app.py

클라우드에서 Streamlit (권장)

Streamlit Community Cloud에 이 GitHub 저장소를 연결하고 Main file을 app.py로 지정하세요. (의존성은 루트 requirements.txt를 사용합니다.)

배포: Vercel — Root Directory를 저장소의 vercel-site로 지정하거나, 해당 폴더에서 vercel CLI로 배포하세요.
